# 🏦 Bank Management System

A Python-based Bank Management System built using Object-Oriented Programming (OOP) principles.

This project simulates a simple banking system where users can create customers, open accounts, perform transactions, and manage their banking data.

## ✨ Features

- Add new customers
- Prevent duplicate national IDs
- Show customer information
- Create bank accounts
- Deposit money
- Withdraw money
- Transfer money between accounts
- Show customer accounts
- Close accounts
- Remove customers
- Save and load banking data using JSON

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- JSON file handling
- Git & GitHub

Clone the repository.

Open the project folder.

Run:
python main.py

📌 Notes
